# Changelog

Todas las modificaciones notables a este proyecto se documentarán en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/lang/es/).

## [0.1.0] - 2025-04-26

### Añadido (Added)
- Script principal de monitoreo con lectura de 4 sensores
- Máquina de estados NORMAL/ADVERTENCIA/PELIGRO
- Integración con ThingSpeak para visualización remota
- Drivers para DS18B20, JSN-SR04T, MQ-4 y MPX5100DP
- Documentación completa en español
- Estructura de repositorio open source

### Pendiente (To Do)

- Implementación de dashboard web personalizado
- Integración con panel solar para alimentación autónoma
- Sistema de alertas SMS/WhatsApp
- Aplicación móvil complementaria
